# Ejercicio 01: Estructura de Estudio Indie

*Nombre:* Irma Arias

## Explicación
Este proyecto organiza los recursos necesarios para un equipo de desarrollo de videojuegos de acción y aventura, separando el código, los assets multimedia y la documentación.

## Árbol de carpetas
- src/: Código fuente principal del juego.
- assets/: Recursos multimedia (imágenes, audio y mapas).
- docs/: Documentación técnica.
- tests/: Pruebas unitarias para validar el código.

## Validación
La estructura fue creada utilizando comandos de terminal (mkdir) para asegurar la jerarquía requerida.
