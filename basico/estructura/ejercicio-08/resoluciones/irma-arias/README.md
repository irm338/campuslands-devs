# Proyecto: Menú Urbano

Este proyecto organiza la gestión de pedidos de comida artesanal.

## Estructura
- `/menu`: Contiene el archivo `platos.json` con el catálogo.
- `/docs`: Contiene las políticas de precios.
- `/assets`: Reservado para imágenes de los productos.
